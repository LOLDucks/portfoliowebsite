3D Printer Upgrade - Klipper on CR-10

I upgraded my Creality CR-10 by installing Klipper firmware to achieve faster, more stable, and more precise prints. The control system is based on a BIGTREETECH SKR Mini E3 V3.0 board and a Raspberry Pi 5, with web-based management via Fluidd, which also supports live webcam streaming.

Main Modifications:

  • Replaced mainboard with BIGTREETECH SKR Mini E3 V3.0
  • Added Raspberry Pi 5 for remote management through Fluidd and Klipper
  • Installed external power supply and fans for improved cooling
  • Repurposed a plastic box as a custom control unit: removed original display, ensured proper airflow, and installed BTT board
  • Added second motor for the Z-axis
  • Reinforced the frame and added anti-vibration feet
  • Modified extruder and cooling fan
  • Integrated BLTouch for auto bed leveling

Why this project:

The goal was to create a quieter, faster, and modular machine, perfect for rapid prototyping. This upgrade also gave me the opportunity to deepen my knowledge of Klipper, GPIO pin management, and interfacing between microcontroller and Raspberry Pi.

Challenges Faced:

  • Firmware flashing issues: Windows added hidden folders to the SD card that broke the update. Solved using Linux to clean the card.
  • Lack of compatible cables: hand-built custom wiring with crimping and soldering.

Pentest GPT - Custom ChatGPT for CTF & Penetration Testing

This project is a customized version of ChatGPT, designed specifically to assist with penetration testing, Capture The Flag (CTF) challenges, and debugging. It's tailored for intermediate users aiming to boost their efficiency, autonomy, and practical understanding.

Main Features:

  • Follows a strict, minimalist workflow: no unnecessary text, no skipped steps
  • Explains every command clearly, but only when needed
  • Waits for user confirmation before proceeding to the next step
  • Maintains session state and avoids repetition
  • Always provides official sources (CVE, docs, articles) for exploits and tools
  • Operates under the assumption of a Kali Linux / VirtualBox setup
  • Supports OSINT, password list generation, scan strategy, and live debugging

Why this project:

I built this GPT to support my daily workflow in CTF and real-world pentests. It helps structure actions, maintain context, and reduce the mental load during complex engagements—while still requiring full user control over the process.

Work in Progress - ESP32 Drone

This custom drone project is built using two ESP32-S3 modules — one dedicated to camera streaming and the other for flight control. The goal is to develop a low-cost, modular drone platform with real-time telemetry and video feed.

Main Components:

  • Motors: RS2205 2300KV brushless motors
  • Battery: 4S 3300mAh 50C LiPo
  • Flight sensors: TOF400C (distance), PMW3901 (optical flow), HMC5883 (magnetometer)
  • Microcontroller: SP32-S3 for control, ESP32-CAM for camera
Work in Progress - EDC Pentesting Tools

This custom-built EDC (Every Day Carry) device combines multiple ESP32-based pentest tool into a single portable toolkit for wireless security research and experimentation.

Key Features:

  • Dual nRF24L01 Modules: Enables BLE and Bluetooth jamming, as well as BLE advertisement spoofing, inspired by tools like RF-Clown.
  • ESP32-WROOM-32U: Equipped with an external antenna to extend Wi-Fi range, supporting functionalities similar to the ESP32 Marauder, including Wi-Fi scanning, deauthentication attacks, and packet injection.
  • CC1101 Sub-GHz Transceiver: Facilitates scanning, capturing, and replaying signals in sub-1 GHz frequencies (e.g., 315, 433, 868, 915 MHz), useful for interacting with devices like garage doors and remote key fobs.
  • Infrared (IR) LEDs: Allows for IR signal transmission to control or interfere with IR-based devices.

Purpose:

The goal of this project is to create a versatile, portable device that consolidates multiple wireless testing tools into one. It's intended for educational purposes, allowing for the exploration and understanding of various wireless communication protocols and their potential vulnerabilities.

Work in Progress - Pwnagotchi

This project is a build of Pwnagotchi, an AI-powered tool designed to automate the collection of WPA/WPA2 handshakes by passively listening to WiFi traffic. It uses bettercap under the hood and evolves over time, optimizing its behavior to capture more handshakes efficiently.

Hardware Setup:

  • Raspberry Pi Zero W as the main board
  • eInk display for minimalist output and status
  • External WiFi adapter (monitor mode capable) for extended range and reliability
  • Portable battery pack for mobile operations

Features:

  • AI-assisted learning to improve capture efficiency
  • Captures WPA/WPA2 handshakes in real time
  • Fully automated, headless operation
  • Custom config for plugins, auto-save, and status messages

The Pwnagotchi interacts with its environment, adapting its scan and deauth strategy based on the networks detected. Ideal for pentesting training, wardriving, or offline password auditing.
